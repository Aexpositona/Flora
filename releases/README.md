# Releases del Juego Flora

Este directorio contiene las versiones compiladas del juego para su distribución.

## Estructura de archivos para cada release:

- `flora.exe` - Ejecutable principal del juego
- `plantas.json` - Base de datos de plantas
- `imagenes/` - Carpeta con las imágenes de las plantas
- `INSTRUCCIONES_PROFESORES.txt` - Guía para educadores
- `README_RELEASE.txt` - Instrucciones de uso

## Cómo generar un nuevo release:

1. Ejecuta `generar_exe.bat` desde la raíz del proyecto
2. El script generará automáticamente la carpeta `flora_release/`
3. Comprime la carpeta y súbela como nuevo release
